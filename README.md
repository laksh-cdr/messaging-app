# Real Time Messaging App
This a full-stack, real time, messaging app that uses
- Node.js
- Socket.io
- Express.js
- Mongodb
to make the app work.

## Features
The app currently has features listed below:
- Sign up/login authentication with a unique username
- Conversational style messaging interaction.
- Multiple users can be added to a single conversation using a unique conversation ID, which you can send to other users so that they can join the conversation.
- Creating a new conversation creates a unique key for that conversation.
- There are other things you can see when you open the source code.

## Note:
Make sure to run npm install after downloading the source code to download the node_modules, as it is not included in the repository.
